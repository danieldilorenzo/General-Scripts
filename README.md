# General Scripts

Coleção pessoal de scripts de automação e tweaks para Games, Linux e Windows 🎮🛠️✨

_Personal collection of automation scripts and system tweaks for Gaming, Linux, and Windows 🎮🛠️✨_

<br>

## About scritps


- [ ] [ ```disk_space.py``` ](./Scripts/disk_space.py)

Script python que verifica se um armazenamento conectado é HDD, SSD ou NVMe. Mostra também espaço total, espaço livre, ocupado e porcentagem do espaço ocupado. Compatível com Linux e Windows.

_Python script that checks if a connected storage device is HDD, SSD, or NVMe. It also displays total space, free space, used space, and the percentage of space used. Compatible with Linux and Windows._

<br>


- [ ] [ ```monitor_preços_steam.py``` ](./Scripts/monitor_precos_steam.py)

Script python que verifica o preço dos jogos na Steam, mostra se está em desconto, e qual o preço normal dele.

_A Python script that checks the price of games on Steam, shows if they are on sale, and what their regular price is._

## About General Tweaks

- [ ] [```desabilitar_transferencia_de_dados_usb.md```](./General_Tweaks/desabilitar_transferencia_de_dados_usb.md)

Passo a passo para impedir que o usuário consiga acessar o pendrive no Windows, incluindo visualização de conteúdo do pendrive, visualização, remoção e exclusão de arquivos.

_Step-by-step instructions to prevent the user from accessing the USB drive in Windows, including viewing the drive's contents, viewing, removing, and deleting files._

<br>

- [ ] [```impedir_o_usuario_de_interagir_com_o_microsoft_defender_antivirus.md```](./General_Tweaks/desabilitar_transferencia_de_dados_usb.md)

Passo a passo para impedir o usuário de interagir com o Microsoft Defender Antivirus, impedindo que ele desative o scan e busca de ameaças, o que o deixaria vulnerável.

_Step-by-step instructions to prevent the user from interacting with Microsoft Defender Antivirus, preventing them from disabling threat scanning and searching, which would leave them vulnerable._
